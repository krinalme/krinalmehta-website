# Internal Linking Metrics and KPIs

Framework for measuring the effectiveness of semantic internal linking strategies.

## Core Metrics

### 1. Crawl Depth
- **Target**: Important pages within 3 clicks of homepage
- **Impact**: Pages within 1-3 clicks receive **9x more SEO traffic**

### 2. Internal Link Count
- Minimum: 1 link per page (no orphans)
- Important pages: 10+ inbound links
- Warning: 50+ outbound links = over-optimization risk

### 3. Anchor Text Distribution
| Type | Target % |
|------|----------|
| Exact Match | 10-20% |
| Partial Match | 20-30% |
| Natural/Descriptive | 30-40% |
| Branded | 10-20% |
| Generic | 5-10% |

## SEO Performance Metrics

### Organic Traffic Growth
- Measure traffic to pages after adding internal links
- Expected: 15-30% increase within 6-8 weeks
- Track via GA4 landing page report

### Ranking Improvements
- Monitor target keywords in GSC
- Expected: 2-5 position improvement for supported pages
- Track cluster-level keyword counts

### Indexation Rate
- Pages indexed / Total pages
- Target: >95% for important content
- Check via `site:` search or GSC Coverage

## AI/AEO Metrics

### AI Citation Tracking
- Monitor brand mentions in ChatGPT, Perplexity, Google AI Overviews
- Tools: manual searches, brand monitoring services
- Target: Citations for cluster topics you've built authority on

### Structured Data Validation
- Test breadcrumb schema in Google Rich Results Test
- Verify RelatedLink properties render correctly
- Monitor Search Console enhancements report

## Audit Checklist

**Weekly**
- [ ] Check for new orphan pages
- [ ] Review pages with <3 inbound links
- [ ] Monitor broken link reports

**Monthly**
- [ ] Full crawl depth analysis
- [ ] Anchor text distribution review
- [ ] Link equity flow visualization
- [ ] AI citation spot checks

**Quarterly**
- [ ] Complete internal link audit
- [ ] Topic cluster performance review
- [ ] Competitive gap analysis
- [ ] Strategy adjustment based on data
